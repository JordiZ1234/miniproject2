Tecnologias utilizadas

Para la creación de esta pàgina se ha utilizado las siguientes tecnologias

  • Visual Studio Code

  • HTML

  • CSS

  • Live Server

Visual Studio Code: Este editor de codigo es el que se ha utilizado para crear toda la pàgina web, y construir todos los codigos necesarios para su funcionamiento

HTML: se ha diseñado un archivo HTML para cada jugador y apartado de la pagina, contando con un total de 14 archivos html, gracias a esta tecnologia se ha conseguido construir la estructura base de la pàgina

CSS: Gracias al CSS se ha logrado dar un estilo a toda la estructura de la pàgina web, consigueiendo un dinamismo y una estetica relacionada con las cartas de los jugadores. Se ha utilizado para toda la estetica de las imagenes, del html etc... Como por ejemplo: display-flex, para que los objetos se puedan mover, los tamaños de las imagenes, cambiar los margenes, quitar o poner relleno, generar un espacio entre objetos, configurar los tamaños de los contenedores, y mucho mas.

Live Server: Con el Live Server se ha podido visualizar el contenido html, por ende se ha conseguido ver los cambios en tiempo real y poder ir comprobando que todo el funcionamiento de los codigos sea el previsto
